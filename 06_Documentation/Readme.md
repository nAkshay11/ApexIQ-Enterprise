# Enterprise Sales Analytics Platform

## Project Overview

The Enterprise Sales Analytics Platform is an end-to-end Data Analytics project built using PostgreSQL, Python ETL, SQL, and Power BI.

This project simulates a real-world enterprise sales environment by integrating customer, employee, product, order, payment, and inventory data into a centralized PostgreSQL database.

The data is extracted from CSV files, transformed using Python, loaded into PostgreSQL, analyzed using SQL, and visualized through interactive Power BI dashboards.
